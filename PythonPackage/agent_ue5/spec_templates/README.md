# Spec Templates

## 1. 目录用途

本目录存放 **通用 Spec 模板**，用于给宿主项目生成项目实例 Spec 的起点。

这里放的是模板，不是项目实例。

## 2. 应放在这里的内容

- 通用场景模板
- 通用语义字段模板
- 可复用的最小示例模板

## 3. 不应放在这里的内容

- 某个宿主项目的具体关卡布局
- 某个项目的测试记录
- 项目专属资源路径
- 项目运行产物

## 4. 与宿主项目的关系

建议工作流：

1. 从本目录选模板
2. 复制到宿主项目的 `AgentSpecs/`
3. 在宿主项目中补齐地图、资产、Actor、约束等项目语义

宿主项目实例 Spec 应放在宿主项目仓，例如：

```text
<HostProject>/
└── AgentSpecs/
    ├── levels/
    └── gameplay/
```
